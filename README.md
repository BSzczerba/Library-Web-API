# Library-Web-API
My first Web API in ASP.NET Core 6.
![LibraryApi](https://github.com/BSzczerba/Library-Web-API/assets/129983724/ef9a8f87-d415-443d-b5a4-49a1e002b27b)
# Using
* EF Core 6.0
* ASP.NET Core Web API
* AutoMapper 
* SWAGGER
* Nlog
* FluentValidation
* JwtBearer
